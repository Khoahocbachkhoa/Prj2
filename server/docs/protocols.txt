Đăng ký và xác thực:

1. Đăng ký người dùng:
    Mô tả: 
        Client gửi yêu cầu đăng ký kèm tài khoản + mật khẩu tới server.
    request:
        REGISTER <username> <password>
    response:
        200 REGISTER_SUCCESS
            Đăng ký thành công
        409 USER_EXISTS
            Tên đăng nhập đã tồn tại
        400 BAD_REQUEST 
            Sai định dạng
        422 WEAK_PASSWORD
            Mật khẩu quá yếu
        500 ERROR_SERVER
            Nếu phía server có lỗi gì đó
    Ghi chú:
        1a. Password đủ mạnh bao gồm ít nhất 10 ký tự, có chữ in hoa, in thường, chữ số và ký tự đặc biệt

2. Đăng nhập:
    Mô tả:
        Client đăng nhập tới server để truy cập file của mình, chia sẻ hoặc nhận file của người khác
        hoặc kết nối tới 1 workspace
    request:
        LOGIN <username> <password>
    response:
        200 LOGIN_SUCCESS
            Đăng nhập thành công
        401 INVALID_CREDENTIALS
            Sai tên đăng nhập hoặc mật khẩu
        402 ALREADY_LOGIN
            Đã đăng nhập rồi, đề nghị đăng xuất
        400 BAD_REQUEST
            Sai định dạng
        500 ERROR_SERVER
            Nếu phía server có lỗi gì đó

3. Đăng xuất
    Mô tả:
        Client đăng xuất khỏi phiên
    request:
        LOGOUT
    response:
        200 LOGOUT_SUCCESS
        201 NOT_LOGIN_YET

Điều hướng và thao tác:

4. In ra thư mục làm việc
    request:
        PWD
    response:
        200 PATH
            Trả về mã 200 Oke và tên thư mục hiện tại
        401 NOT_LOGIN_YET
            Nếu người dùng chưa đăng nhập
        500 ERROR_SERVER

5. Liệt kê nội dung thư mục
    request:
        LIST
    response:
        200 LIST_BEGIN
            <... contents ...>
        200 LIST_END\r\n
            Trả về nội dung có trong thư mục
        401 NOT_LOGIN_YET
            Nếu người dùng chưa đăng nhập
        402 FOLDER_DELETED
            Nếu folder đã được đánh dấu là xóa
        404 FOLDER_NOT_FOUND
            Nếu server không tìm thấy folder hiện tại
        500 ERROR_SERVER
    Ghi chú: Chỉ hỗ trợ hiển thị thư mục hiện tại, chưa hỗ trợ hiển thị path cụ thể

6. Điều hướng
    request:
        CD path
    response:
        200 CHANGE_DIRECTORY_SUCCESS
            Điều hướng thành công
        401 NOT_LOGIN_YET
            Chưa đăng nhập
        403 ACCESS_DENIED
            Không có quyền truy cập
        422 INVALID_PATH
            Không có path hoặc path không hợp lệ
        409 NOT_A_DIRECTORY
        500 ERROR_SERVER
    Ghi chú: Chỉ hỗ trợ hiển thị điều hướng trong thư mục hiện tại tới 1 con hoặc .., chưa hỗ trợ điều hướng tới path cụ thể

7. Tạo thư mục mới
    request:
        MKDIR new_folder
    response:
        200 MKDIR_SUCCESS
            Thành công
        401 NOT_LOGIN_YET
            Chưa đăng nhập
        403 ACCESS_DENIED
            Không có quyền tạo thư mục
        409 FOLDER_ALREADY_EXISTS
            Tạo một thư mục trùng tên thư mục đã tồn tại
        422 INVALID_FOLDER_NAME
            Tên thư mục không hợp lệ
        500 ERROR_SERVER

8. Xóa 1 file hoặc 1 thư mục
    request:
        DELETE path_or_file_name
    reponse:
        200 DELETE_SUCCESS
            Thành công
        401 NOT_LOGIN_YET
            Chưa đăng nhập
        403 ACCESS_DENIED
            Không có quyền
        404 RESOURCE_NOT_FOUND
            Không tồn tại tài nguyên
        409 DIRECTORY_NOT_EMPTY
            Không được xóa 1 thư mục không rỗng
        500 ERROR_SERVER

9. Đổi tên
    request:
        RENAME old_name new_name
    response:
        200 RENAME_SUCCESS
            đổi tên thành công
        401 NOT_LOGIN_YET
        403 ACCESS_DENIED
            Chỉ được phép đổi tên nếu chủ sỡ hữu là mình
        404 FILE_NOT_FOUND
        500 ERROR_SERVER

Truyền tải:

10. upload
    request:
        UPLOAD <filename> <filesize>
    reponse:
        150 READY_TO_RECEIVE
            Cho phép client bắt đầu upload file, khi upfile xong gửi lại phản hồi
        200 UPLOAD_SUCCESS
            Báo hiệu server nhận file thành công
        401 NOT_LOGIN_YET
            Chưa đăng nhập
        403 ACCESS_DENIED
            Không có quyền
        409 FILE_ALREADY_EXISTS
            File đã tồn tại
        422 INVALID_FILE_NAME
            Tên file không hợp lệ
        500 ERROR_SERVER
            Lỗi từ server

11. download
    request:
        DOWNLOAD <filename>
    reponse:
        150 START_DOWNLOAD <filesize>
            Server sẵn sàng bắt đầu gửi file
        200 DOWNLOAD_SUCCESS
            Server gừi file xong
        401 NOT_LOGIN_YET
            Chưa đăng nhập
        403 ACCESS_DENIED
            Không có quyền
        404 FILE_NOT_FOUND
            Không tìm thấy file
        500 ERROR_SERVER
            Lỗi từ server

Chia sẻ:
12. Chia sẻ 1 file 
    Mô tả:
        Chia sẻ 1 file cho 1 user khác để user kia có thể tải về
    request:
        SHARE_FILE <filename> <username>
    reponse:
        200 SHARE_FILE_SUCCESS
            Chia sẻ thành công
        401 NOT_LOGIN_YET
            Chưa đăng nhập
        403 ACCESS_DENIED
            Không có quyền chia sẻ file này
        404 FILE_NOT_FOUND
            Không tìm thấy file
        404 USER_NOT_FOUND
            Không tìm thấy user được chia sẻ
        409 FILE_ALREADY_SHARED
            File đã được chia sẻ cho user này rồi
        422 INVALID_USERNAME
            Username không hợp lệ
        500 ERROR_SERVER
            Lỗi phía server

13. Chia sẻ 1 thư mục làm việc chung
    Mô tả:
        Chia sẻ 1 folder cho các user khác để có thể upload, download chung
    request:
        SHARE_FOLDER <foldername> <username> <role>
    response:
        200 SHARE_FOLDER_SUCCESS
            Chia sẻ thư mục thành công
        401 NOT_LOGIN_YET
            Chưa đăng nhập
        403 ACCESS_DENIED
            Không có quyền chia sẻ thư mục này
        404 FOLDER_NOT_FOUND
            Không tìm thấy thư mục
        404 USER_NOT_FOUND
            Không tìm thấy user được chia sẻ
        409 FOLDER_ALREADY_SHARED
            Folder đã được chia sẻ cho user này rồi
        422 INVALID_ROLE
            Role không hợp lệ
        500 ERROR_SERVER
            Lỗi từ phía server

14, Hủy quyền chia sẻ file
    request:
        UNSHARE_FILE <filename> <username>

    response:
        200 UNSHARE_FILE_SUCCESS
            Hủy chia sẻ file thành công
        401 NOT_LOGIN_YET
            Chưa đăng nhập
        403 ACCESS_DENIED
            Không có quyền hủy chia sẻ file này
        404 FILE_NOT_FOUND
            Không tìm thấy file
        404 USER_NOT_FOUND
            Không tìm thấy user
        404 SHARE_NOT_FOUND
            File chưa được chia sẻ cho user này
        500 ERROR_SERVER
            Lỗi phía server

15. Hủy chia sẻ thư mục làm việc chung
    request:
        UNSHARE_FOLDER <foldername> <username>

    response:
        200 UNSHARE_FOLDER_SUCCESS
            Hủy chia sẻ thư mục thành công
        401 NOT_LOGIN_YET
            Chưa đăng nhập
        403 ACCESS_DENIED
            Không có quyền hủy chia sẻ thư mục này
        404 FOLDER_NOT_FOUND
            Không tìm thấy thư mục
        404 USER_NOT_FOUND
            Không tìm thấy user
        404 SHARE_NOT_FOUND
            Folder chưa được chia sẻ cho user này
        500 ERROR_SERVER
            Lỗi phía server

16. Liệt kê danh sách user được chia sẻ

    Mô tả:
        Hiển thị danh sách các user đang được chia sẻ quyền
        truy cập tới file hoặc thư mục

    request:
        LIST_SHARED_USERS_FILE <resource>
        LIST_SHARED_USERS_FOLDER <resource>
    response:
        200 LIST_SHARED_USERS_BEGIN
            <username> <role>
            <username> <role>
            ...
        200 LIST_SHARED_USERS_END
            Liệt kê thành công
        401 NOT_LOGIN_YET
            Chưa đăng nhập
        403 ACCESS_DENIED
            Không có quyền xem danh sách chia sẻ
        404 RESOURCE_NOT_FOUND
            Không tìm thấy file hoặc thư mục
        500 ERROR_SERVER
            Lỗi phía server

17. Xem danh sách tài nguyên được chia sẻ với tôi
    Mô tả:
        Hiển thị các file và thư mục mà user khác đã chia sẻ
        cho người dùng hiện tại
    request:
        LIST_SHARED_WITH_ME
    response:
        200 LIST_SHARED_WITH_ME_BEGIN
            FILE <filename> <owner>
            FOLDER <foldername> <owner> <role>
            ...
        200 LIST_SHARED_WITH_ME_END
            Liệt kê thành công
        401 NOT_LOGIN_YET
            Chưa đăng nhập
        500 ERROR_SERVER
            Lỗi phía server

18. Tải file được chia sẻ với tôi
    request:
        DOWNLOAD_SHARED <username> <filename>
    response:
        Tương tự Download

19. Di chuyển vào thư mục được chia sẻ
    request:
        OPEN_SHARED_FOLDER <owner> <folder>
    reponse:
        200 Ok
            điều hướng thành công
        401 ACCESS_DENIED
            không có quyền
    Ghi chú:
        Khi cd vào 1 thư mục được chia sẻ, các lệnh download, upload, .. dùng bình thường

20. Thoát khỏi chế độ chia sẻ
    request:
        EXIT_SHARED
    response:
        200 Ok
            Thoát khỏi chế độ chia sẻ.
